📄 Guia Rápido de Tags HTML

As tags mais usadas em sites simples — resumido e direto ao ponto.

<header>

Cabeçalho da página. Fica no topo e geralmente contém o logo e o título do site.

<header> <h1>Meu Site</h1> </header>

<nav>

Barra de navegação. Contém os links para as seções ou páginas do site.

<nav> <a href="#">HOME</a> <a href="#">CONTATO</a> </nav>

<main>

Conteúdo principal da página. É o "miolo" do site — textos, imagens, tabelas.

<main> <p>Conteúdo aqui...</p> </main>

<section>

Divide o conteúdo em blocos com o mesmo tema. Ex: seção de livros, seção de contatos.

<section id="livros"> <h2>Livros</h2> </section>

<div>

Caixa genérica para agrupar elementos. Não tem significado semântico, mas é muito usada para organizar o layout com CSS.

<div class="card"> <p>Conteúdo agrupado</p> </div>

<a>

Link. O href define o destino. Use target="_blank" para abrir em nova aba.

<a href="https://google.com" target="_blank">Google</a> <!-- Para ir a uma seção da mesma página: --> <a href="#contatos">Ir para Contatos</a>

<img>

src é o caminho da imagem. alt é o texto alternativo. Não tem tag de fechamento!

<img src="foto.jpg" alt="Descrição da foto" width="200">

<table> <tr> <th> <td>

tr = linha | th = célula de título (negrito) | td = célula de dado normal.

<table border="1"> <tr> <th>Nome</th> <th>Email</th> </tr> <tr> <td>João</td> <td>joao@email.com</td> </tr> </table>

<ul> e <li>

ul = lista com bolinhas | ol = lista numerada | li = cada item da lista.

<ul> <li>HTML</li> <li>CSS</li> <li>JavaScript</li> </ul>

<footer>

Rodapé da página. Fica no final e geralmente tem créditos ou informações extras.

<footer> <p>© 2024 Meu Site</p> </footer>

id="" vs class=""

id = único na página, selecionado com # no CSS. class = pode repetir, selecionado com . no CSS.

<!-- HTML --> <div id="unico"></div> <div class="card"></div> /* CSS */ #unico { color: red; } .card { color: blue; }

display: flex — o mais importante do CSS!

Coloca elementos lado a lado e facilita o alinhamento. Resolve 90% dos problemas de posicionamento.

nav { display: flex; justify-content: center; /* alinha horizontal */ align-items: center; /* alinha vertical */ gap: 20px; /* espaço entre os itens */ }
